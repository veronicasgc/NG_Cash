# CT-001 Login com sucesso

## Objetivo
Validar que um usuário cadastrado consegue realizar login.

## Pré-condição
Usuário previamente cadastrado na base de dados.

## Passos
1. Enviar username válido.
2. Enviar password válido.
3. Executar endpoint POST /login.

## Resultado esperado
- Status Code 200.
- Token JWT retornado.
- Usuário autenticado com sucesso.

## Resultado obtido

Status Code: 200 OK

Token JWT retornado com sucesso.
